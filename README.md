# waterfilm_segmentation
代码可用于路面积水区域的分割以及水膜覆盖率计算


将待检测的图像存入根目录下的文件夹——“work/input”

然后运行根目录下的 ‘main.py’

即可在对应文件夹中生成对应的积水区域分割图像，倾斜校正图像，以及分块计算水膜覆盖率的图像
